首先，感謝使用這個計分小工具，這裡是說明此工具的手冊，如需關閉手冊請再次點擊下方的help_outline。

這個小工具會自動記憶上一次初始化的資料，所以當下次初始化時會自動填入上次設定的資料。

另外，考量到手機版面有限，可以使用小工具右下角的menu來顯示或隱藏下方的選單。為了方便使用，選單的是否隱藏也會自動記憶。

此小工具的原始碼可以在 GitHub 上看到。點擊下方的home可以查看我的其他 Projects、作品、小工具等。
